Aula 7 - Operadores

Operadores podem ser:

* Aritméticos
* Atribuição
* Relacionais
* Lógicos
* Ternário

Exercitanto ordem de precedência

var a = 5 + 3
var b = a % 5
var c = 5 * b ** 2
var d = 10 - a / 2
var e = 6 * 2 / d
var f = b % e + 4 / e

Auto-atribuição

var n = 3
n = n + 4 ou n += 4
n = n - 5 ou n -= 5
n = n * 4 ou n *= 4
n = n / 2 ou n /= 2
n = n ** 2 ou n **= 2
n = n % 5 ou n %= 5

Incremento

var x = 5
x++
++x
x--
--x

Relacionais

> - maior
< - menor
>= - maior ou igual
<= - menor ou igual
== - igualdade
!= - difereça
Obs: Nas operações sempre retornam um valor "boolean"

Identidade

=== - igual restrito ou identico
!== - desigual restrito

Lógicos

! - negação
&& - conjunção(e)
|| - disjunção(ou)
Exemplos:
idade >= 15 && idade <= 17 - a idade está entre 15 e 17?
estado == 'RJ' || estado == 'SP' - o estado é RJ ou SP?
salário > 1500 && sexo != 'M' - salário é maior que 1500 quando mulher?

Ternário

?
:
Estrutura:
teste ? true : false
média >= 7.0 ? "APROVADO" : "REPROVADO"
